# funds
## 过滤条件
1.过滤低收益的类型'货币型'、 '固定收益'、'定开债券'、'QDII'、'理财型'、'QDII-指数'、'分级杠杆'。        
2.基金运行时间大于俩年。        
3.标准差小于4，并且夏普比率大于0.1。        
4.2012年至今，历史年收益都大于-5.0。        
5.季度涨幅和年度涨幅的排名在良好一下的占比。
